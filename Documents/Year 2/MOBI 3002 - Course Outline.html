MOBI3002

Course Weight: 60.00
 

Course Description
Android provides a platform; a single application model, tools, and an open marketplace (for distribution) that lets
you deploy applications to users across a wide range of mobile devices. In this course, learners will develop mobile
software applications that run on the Android platform .
 

Rationale
With the growing popularity of smart phones and tablet computing, the demand for fully-featured applications deployed on these devices is rapidly increasing. Whether it be specially-designed applications or simply building mobile-friendly versions of existing software or websites, the ability to work in a mobile environment is critical to application developers.


Prerequisite(s):
PROG 1400 (Introduction to Object Oriented Programming)
 

Corequisite(s):
None


Learning Outcomes
Learners are required to successfully complete each course outcome. In keeping with NSCC’s approach to portfolio learning, learners will have demonstrated the ability to:
 

  1. Configure the required tools (e.g. IDE, SDK, Emulator) to develop Android applications, differentiating
    between platform versions and terminology unique to the Android operating system.

  2. Design mobile application user interfaces for a variety of mobile devices with a focus on usability.2.

  3. Demonstrate various approaches to achieve data persistence in Android applications.3.

  4. Deploy Android applications on both physical and virtual devices and explore the approaches to distribution.4.

  5. Practice appropriate object oriented design, testing and debugging procedures in the development of Android applications.

  6. Produce professional level communication documents and presentations using an office software suite.

Grading: The pass for this course is 60%
